# Air-quality-data-analysis
Analysis of air pollution data using Python to identify chemical trends
## Objectives
 Analyze air pollution data to identify trends in chemical pollutants.
 ## Tools used
 -Python
 -Pandas 
 -Matplotlib
## Dataset
Air Quality dataset containing CO, NO2,Benzene levels
## Key Insights Fron Visualization
1. ** CO Trend Over Time **
- CO fluctuate significantlt across time periods.
- It  indicate varying pollution intensity , possibly due to traffic or industrial activity.
2. ** Pollution Comprison (CO , NO2, BENZENE) **
  - NO2 is primary driver of air quality fluctuations, showing sharp peaks and high variability while
    benzene and CO remain comparatively stable
  - This suggest that pollution control stratigies should prioritize reducing NO2 emission to
    acheive improvement
 3. ** Temperature vs CO analysis **
  - CO is largely independent of temperature variations, highlighting the importance of human-driven
    emission sources over environmental conditions in determining air quality.
## Conclusion
This project demonstrate basic data cleaning, analysis, and visualization using Python.
